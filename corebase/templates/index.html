Bienvenido

Este sistema es un programa de prueba cuya función es ayudar en el proceso de autenticación con firma digital bajo un esquema moderno.
Considere que no es un sistema de producción sino de pruebas.

Para poder crear un proceso de inicio de sesión con firma digital se necesita tener registrada una institución y sus respectivas URLs de notificación (ver opción editar de institución).
Una vez creada la institución se podrá acceder a los detalles de la misma para observar las claves de accesos.
Se proporciona lo siguiente en formato PEM:

  • private key: Clave privada para desencriptar mensajes.
  • server public key: Clave pública para encriptar mensajes del servidor.
  • public certificate: Certificado de validación de servicio.
Nota: Mantenga esta información en un lugar seguro.
